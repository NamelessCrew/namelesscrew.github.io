## 🎯 立即上线（5 分钟完成）

### 第一步：更新配置（1分钟）⚠️ 重要

**必须替换**：将以下文件中的 `yourusername` 替换为你的 GitHub 用户名

1. **index.html**（约10处）
2. **sitemap.xml**（1处）  
3. **robots.txt**（1处）

**快速替换方法：**

方法一：使用命令（macOS）
```bash
cd ~/my-homepage
find . -type f \( -name "index.html" -o -name "sitemap.xml" -o -name "robots.txt" \) -exec sed -i '' 's/yourusername/你的实际用户名/g' {} +
```

方法二：手动替换（推荐，更安全）
- 用编辑器打开这3个文件
- 搜索 "yourusername"
- 全部替换为你的 GitHub 用户名

### 第二步：创建 GitHub 仓库（2分钟）

1. 访问：https://github.com/new
2. 仓库名：**你的用户名.github.io**
   - 例如：`universalmariner.github.io`
3. 勾选 **Public**（必须公开）
4. **不要**勾选 "Add a README"
5. 点击 "Create repository"

### 第三步：部署（2分钟）

```bash
cd ~/my-homepage
./deploy.sh

# 按提示输入仓库地址
# 例如：https://github.com/universalmariner/universalmariner.github.io.git
```

### 第四步：启用 GitHub Pages（1分钟）

1. 进入你的仓库
2. 点击 **Settings** → **Pages**
3. Source 选择 **main** 分支
4. 点击 **Save**
5. 等待 2-5 分钟

### ✅ 完成！

访问：`https://你的用户名.github.io`

## 🔍 让搜索引擎收录（重要！）

### Google Search Console（最重要）

1. **注册验证**
   - 访问：https://search.google.com/search-console/
   - 添加资源：`https://你的用户名.github.io`
   - 选择 **HTML 标签**验证方法
   - 复制验证代码

2. **添加验证代码**
   - 在 `index.html` 的 `<head>` 部分添加验证代码
   - 示例：
   ```html
   <meta name="google-site-verification" content="你的验证码" />
   ```

3. **推送更新并验证**
   ```bash
   cd ~/my-homepage
   git add .
   git commit -m "Add Google verification"
   git push
   ```
   - 回到 Search Console 点击"验证"

4. **提交站点地图**
   - 在 Search Console 左侧菜单：站点地图
   - 添加新的站点地图：`sitemap.xml`
   - 点击提交

5. **请求索引**
   - 在顶部搜索框输入：`https://你的用户名.github.io`
   - 点击"请求编入索引"

### Bing Webmaster Tools

1. 访问：https://www.bing.com/webmasters/
2. 重复类似 Google 的步骤
3. 提交站点地图

### 百度站长平台（中文搜索）

1. 访问：https://ziyuan.baidu.com/
2. 添加网站并验证
3. 提交链接

**Q: 网站显示 404？**
A: 等待 2-5 分钟，GitHub Pages 需要部署时间。清除浏览器缓存重试。

**Q: 图片不显示？**

**Q: 多久能被搜索到？**
