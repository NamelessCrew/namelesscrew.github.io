================================================================================
                           快速开始 (3步上线)
================================================================================

第 1 步: 预览网站 👀
    cd ~/my-homepage
    ./preview.sh
    # 浏览器访问 http://localhost:8000

第 2 步: 自定义内容 ✏️
    # 编辑 index.html
    # 参考 example.html 的完整示例
    # 参考 CUSTOMIZATION.md 学习如何修改
    
第 3 步: 部署上线 🚀
    ./deploy.sh
    # 按提示输入 GitHub 仓库地址
    # 在 GitHub 设置中启用 Pages
    # 访问 https://你的用户名.github.io
