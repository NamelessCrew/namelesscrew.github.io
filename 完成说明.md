## 🌐 部署到 GitHub Pages

### 步骤：

1. **在 GitHub 创建仓库**
   - 仓库名：`你的用户名.github.io`
   - 设为 Public（公开）

2. **运行部署脚本**
   ```bash
   cd ~/my-homepage
   ./deploy.sh
   ```

3. **启用 GitHub Pages**
   - 进入仓库的 Settings
   - 找到 Pages 选项
   - Source 选择 `main` 分支
   - 点击 Save

4. **访问你的网站**
   - 等待 2-5 分钟
   - 访问：`https://你的用户名.github.io/`
