# 添加占位符图片说明文件

请将以下图片放入 images 文件夹：

1. profile.jpg - 你的个人照片（建议尺寸：500x500 像素，正方形）
2. paper1.jpg - 第一篇论文的配图（建议尺寸：800x600 像素）
3. paper2.jpg - 第二篇论文的配图（建议尺寸：800x600 像素）

## 图片要求

### 个人照片 (profile.jpg)
- 格式：JPG/PNG
- 尺寸：推荐 500x500 像素（正方形）
- 大小：建议小于 200KB
- 说明：清晰的个人照片，背景简洁

### 论文配图 (paper1.jpg, paper2.jpg, ...)
- 格式：JPG/PNG
- 尺寸：推荐 800x600 像素（4:3 比例）
- 大小：建议每张小于 500KB
- 说明：可以是论文的架构图、效果图或代表性图片

## 临时占位符

如果暂时没有图片，可以使用以下占位符服务：
- https://via.placeholder.com/500x500 （个人照片）
- https://via.placeholder.com/800x600 （论文配图）

在 index.html 中可以临时使用：
```html
<img src="https://via.placeholder.com/500x500/2563eb/ffffff?text=Your+Name" alt="Your Name">
```
